# Cybersecurity Career Tracker

A 6-month journey to become a Security Analyst. Study weekdays only, 90 minutes per day.

## Features
- 240 days of structured curriculum
- Watch, Read, Lab, and Notes for each day
- Progress tracking with localStorage
- Google Sheets sync
- Phase tabs and week accordions

## How to Use
1. Open `index.html` in your browser
2. Click on weeks to expand and see daily tasks
3. Click "Mark Done" to track progress
4. All data saves automatically

## Google Sheets Setup (Optional)
1. Create a Google Sheet
2. Go to Extensions > Apps Script
3. Deploy as Web App and copy the URL
4. Update `settings.js` with your URL

## Tech Stack
- HTML5, CSS3, JavaScript
- FontAwesome Icons
- Google Fonts
- LocalStorage for data persistence
